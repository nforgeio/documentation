# Azure Hosting Options

Here are the Azure related hosting options:

```yaml
hosting:
  environment: azure
  azure:
    clientId:                  null
    clientSecret:              null
    cloud:                     "global-cloud"
    defaultDiskSize:           "128 GiB"
    defaultOpenEbsDiskSize:    "128 GiB"
    defaultOpenEbstorageType:  "standard-ssd"
    defaultStorageType:        "standard-ssd"
    defaultVmSize:             "Standard_D4as_v4"
    disableProximityPlacement: false
    domainLabel:               "neon-[UUID]"
    faultDomains:              3
    network:                   {}
    region:                    null
    resourceGroup:             "neon-[CLUSTERNAME]"
    subscriptionId:            null
    tenantId:                  null
    updateDomains:             20
```

        <th>Property</th>
        <th>Description</th>

        <tr>
            <td><b>environment</b></td>

        <tr>
            <td><b>azure</b></td>
            <td>
                `string:` Specifies the Amazon Web Services (AWS) hosting settings.

                <table>

                        <th>Description</th>
                    </thead>
                    <tbody>
                        <tr>

                            <td>
                                `bool:` Optionally installs Harbor. This defaults to `false`.
                            </td>
                        </tr>
                    </tbody>
                </table>
            </td>
        </tr>
    </tbody>
</table>
