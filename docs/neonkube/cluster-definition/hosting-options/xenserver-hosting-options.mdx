# XenServer/XCP-ng Hosting Options

Here are the XenServer/XCP-ng related hosting options:

```yaml
hosting:
  environment:
  aws:
  azure:
  hyperv:
  xenserver: 
```

<table>
	<thead>
		<th>Property</th>
		<th>Description</th>
	</thead>
	<tbody>
		<tr>
			<td><b>hosting</b></td>
			<td>
				`string:` Identifies the target hosting environment, one of **aws**, **azure**, **hyperv**, or **xenserver**.
			</td>
		</tr>
	</tbody>
</table>
