# XenServer/XCP-ng Hosting Options

Here's what the XenServer/XCP-ng related hosting options looks like, with the 
default values or **[required]** for property values that must be specified:

```yaml
hosting:
  environment:
  aws:
  azure:
  hyperv:
  xenserver: 
```

<table>
    <thead>
        <th>Property</th>
        <th>Description</th>
    </thead>
    <tbody>
        <tr>
            <td><b>hosting</b></td>
            <td>
                `string:` Identifies the target hosting environment, one of **aws**, **azure**, **hyperv**, or **xenserver**.
            </td>
        </tr>
    </tbody>
</table>
