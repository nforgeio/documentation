# Installation

## Requirements

### Operating System

* Windows Home
* Windows Professional

Future:
* MacOS
* Linux

### Hardware

* RAM: 16GB
  - Virtual Machine has 8GB
  - NeonKube consumes ~5.5 GB
* CPU: 4
* Disk: 64GB

## Windows

### Installing on Windows

1. Download the installer from [here](https://neonkube.io/download).
2. Navigate to the directory where you downloaded the installer to and run the installer.
3. Follow the prompts on the neon-desktop installer to proceed.

### Uninstalling on Windows

1. From the taskbar, click the Start menu.
2. Go to Settings > Apps > Installed Apps.
3. Find and select Neon Desktop.
4. Click the three dots > Uninstall and click it again when the confirmation appears.
5. Follow the prompts on the neon-desktop uninstaller to proceed.
6. Click Finish when complete.
