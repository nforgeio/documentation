# Installation

To use NEONKUBE, you need to install and run NEONDESKTOP, which is a desktop
application that provides a graphical user interface for NEONKUBE. With
NEONDESKTOP, you can easily configure your cluster settings, deploy and update
your cluster nodes, monitor your cluster health and performance, and access your
cluster resources.

## Requirements

### Operating System

- Windows Professional

Future:

- Windows Home
- MacOS
- Linux

### Hardware

- RAM: 16GB
  - Virtual Machine has 8GB
  - NeonKube consumes ~5.5 GB
- CPU: 4
- Disk: 64GB

## Uninstalling

<Tabs groupId="operating-systems">
  <TabItem value="win" label="Windows">
    <ol>
      <li>From the taskbar, click the Start menu.</li>
      <li>Go to Settings > Apps > Installed Apps.</li>
      <li>Find and select Neon Desktop.</li>
      <li>
        Click the three dots > Uninstall and click it again when the
        confirmation appears.
      </li>
      <li>Follow the prompts on the NEONDESKTOP uninstaller to proceed.</li>
      <li>Click Finish when complete.</li>
    </ol>
  </TabItem>
