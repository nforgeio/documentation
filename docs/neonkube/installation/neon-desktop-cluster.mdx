# Neon Desktop

## Prerquisites

Make sure you have neon-desktop installed. To install, follow the neon-desktop installation instructions [here](./neon-desktop)

## Creating a cluster

1. Right click the neon-desktop icon in the taskbar > Deploy built in cluster
   <Image
     img={require("./assets/neon-desktop/deploy-built-in-cluster.png")}
     style={{ width: 300 }}
   />
2. Select Install as Hyper-V Virtual Machine and click Next
   <Image
     img={require("./assets/neon-desktop/deploy-built-in.png")}
     style={{ width: 300 }}
   />
3. Grab a coffee and wait for neon-desktop to create the cluster.
4. Once the cluster is installed, navigate to the neon-desktop [dashboard](https://neon-dashboard.desktop.neoncluster.io/)
